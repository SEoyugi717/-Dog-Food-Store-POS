# Dog Food Store POS

A specialized Point of Sale and inventory management system for pet retail. This application automates stock tracking, handles sales transactions, and organizes product categories to ensure efficient business operations.

## Tech Stack
* **Language:** PHP
* **Database:** MySQL (XAMPP/phpMyAdmin)
* **Server:** Apache
* **Frontend:** HTML, CSS, JavaScript

## Setup
1. Move the project folder to your XAMPP `htdocs` directory.
2. Open **phpMyAdmin** and create a new database.
3. Import the `.sql` file included in the repository.
4. Verify database connection settings in the config files.
5. Launch the system via `localhost/[folder-name]`.
